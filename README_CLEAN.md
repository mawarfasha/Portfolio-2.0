## 🔧 Customization

All main styling variables are defined in CSS custom properties at the top of `style_clean.css`:

```css
:root {
    --bg-color: #1c1917;
    --accent-color: #f59e0b;
    --text-primary: #ffffff;
    --text-secondary: #e5e7eb;
    /* ... */
}
```

Simply modify these values to change the entire color scheme!
